# Commands

## sdgo version

Prints the CLI version.

## sdgo new <project>

Creates a runnable project from the sdkitgo template repository.

Options:

- `--module`: Go module path.
- `--source`: template source directory.
- `--force`: overwrite existing files.

## sdgo run

Runs the current Go project with the built-in hot reload runner.

Aliases:

- `sdgo dev`

Options:

- `--cmd`: command to execute.
- `--watch`: comma-separated watch paths.

## sdgo gen module <name>

Creates a module skeleton and matching documentation.

Options:

- `--force`: overwrite existing files.
- `--with-docs`: reserved; docs are generated by default.
